## Code example

Removes vowel letters from the string

```
function disemvowel(str) {
  const vowels = ['A', 'E', 'I', 'O', 'U'];
  return str
    .split('')
    .map((el) => vowels.indexOf(el.toUpperCase()) !== -1 ? '' : el)
    .join('')
}
```
